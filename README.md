For å mestre React er det mange ting man skal forstå. Det viktigste å starte med er React sin tilstandsmodell og hvordan
denne forholder seg til tilstanden i nettleseren. Et `<div>`-element i React ser ut som det samme som en `<div>` i
nettlesere, men det er subtile men viktige forskjeller. For å koble sammen de to verdene bruker React hooks som
useState, useEffect og useRef. I denne forestillingen live koder vi med forklaring en demo som viser hvordan
`<dialog>`-elementet som ble innført i HTML i 2022 kan brukes effektiv i React. Gjennom å utforme en applikasjon
gradvis utforsker vi React sin tilstandsmodell, enkle grep for brukervennlighet, kodekvalitet og utviklingsprosess.
